# Regularized Inverse Holographic Volume Reconstruction (RIHVR) for Digital Inline Holography (DIH)
This repository houses the (source) code for Regularized Inverse Holographic Volumetric Reconstruction developed at University of Minnesota (Twin Cities) in Professor Jiarong Hong's lab by Kevin Mallery and Gauresh Raj Jassal. 

For Windows and LINUX installers, along with installation instructions and test data, please see the OSF archive at: https://doi.org/10.17605/OSF.IO/QWRDP

The code is CUDA-accelerated, and further details regarding it can be found in the following papers:

Jassal, Gauresh Raj, Kevin Mallery, and Jiarong Hong. "A Regularized Inverse Holographic Volume Reconstruction (RIHVR) GUI Program for Digital Inline Holography (DIH)." Journal of Open Research Software (2026).

Mallery, Kevin, and Jiarong Hong. "Regularized inverse holographic volume reconstruction for 3D particle tracking." Optics Express 27, no. 13 (2019): 18069-18084.

The authors acknowledge the financial support from the National Science Foundation under Grant No. NSF-MCB-2438892.

# Digital Inline Holography (DIH)
Digital inline holography (DIH) was invented by Hungarian physicist Dennis Gabor, to record the information of a three-dimensional scene onto a two-dimensional image plane -- a hologram, meaning _whole message_. This is achieved by exploiting the wave nature of light to record the two-dimensional interference pattern generated by the interaction of light with the propagation medium and the object field, as shown in the figure below. The goal of this software is to reconstruct the volume field from the recorded holograms with an option to perform particle tracking on the detected objects. The technique has multiple applications in fluid mechanics, microbiology, particle diagnostics, etc.
