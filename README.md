# Regularized Inverse Holographic Volume Reconstruction (RIHVR) for Digital Inline Holography (DIH)
This repository houses the (source) code for Regularized Inverse Holographic Volumetric Reconstruction developed at the University of Minnesota (Twin Cities) in Professor Jiarong Hong's lab by Kevin Mallery and Gauresh Raj Jassal. 

For the latest Windows and LINUX installers, along with installation instructions and test data, please see the OSF archive at: [https://doi.org/10.17605/OSF.IO/QWRDP](https://doi.org/10.17605/OSF.IO/QWRDP)

# Digital Inline Holography (DIH)
Digital inline holography (DIH) was invented by Hungarian physicist Dennis Gabor to record the information of a three-dimensional scene onto a two-dimensional image plane -- a hologram, meaning _whole message_. This is achieved by exploiting the wave nature of light to record the two-dimensional interference pattern generated by the interaction of light with the propagation medium and the object field, as shown in the figure below. The goal of this software is to reconstruct the volume field from the recorded holograms with an option to perform particle tracking on the detected objects. The technique has multiple applications in fluid mechanics, microbiology, particle diagnostics, etc.

# Quick Start Guide
1) Download the Windows Installer from [GDrive - Windows Installer](https://drive.google.com/file/d/14RncCqclnwr_e4DRJBF1Em8ZHwzjZyd9/view?usp=sharing), or the LinuxAppImage from [GDrive - Linux AppImage](https://drive.google.com/file/d/1a8kE3prEmbwEUNIzEmW3pWndnKVZ04V8/view?usp=sharing)
2) Download the _Dandelions_ test case from [GDrive - Dandelion Data](https://drive.google.com/drive/folders/1OSCuNtha1z6xMWWJ4iuyWMh6UmukKK72?usp=sharing) or the _Rotating Rods_ from [GDrive - Rotating Rods](https://drive.google.com/drive/folders/1AGj9QrqHq4RU1cM-mm7gX3QU6MuZb0bb?usp=sharing).
3) Use the following parameters to run the test cases.
